# Коллекция оригиналов сборников заданий и решений всесоюзных и всероссийских олимпиад школьников по химии

До 1996 года отсканированы с бумажных оригиналов, в интернете отсутствуют. С 1996 часть отсканирована, часть конвертирована из сканов в интернете. OCR русский без формул. Если есть что добавить (не позднее 1999 года), свяжитесь со мной: igor_sedov@inbox.ru.

Компиляция 1970-х годов представляла собой брошюру неизвестного происхождения, у которой отсутствовали начальные и конечные страницы, а также несколько страниц в середине. Ее самая первая часть является сокращенной компиляцией материалов 1967-69 гг., а далее материалы 1970-78 гг. приведены без сокращений. Брошюра произведена типографским способом, оригинал-макетом, вероятно, служили исходные брошюры, о чем свидетельствует двойная нумерация страниц. В начале надпись от руки "1978 г."
